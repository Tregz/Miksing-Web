Une application simple pour le mixage multimedia

Projet universitaire

Miksing.com est là version web d'une application mobile multi-platformes pour le mixage multimedia. L'application est un projet étudiant du domaine informatique developpé à l'Université du Québec à Trois-Rivières. Le répertoire du code est publique sur GitHub en swift (avec CoreData) pour iOS et en java (avec SQL) pour Android. Les appareils mobiles synchronisent les données via Firebase de Google.
